# 📚 Text Classification in NLP

This repository contains a collection of Jupyter Notebooks covering fundamental to intermediate concepts in Natural Language Processing (NLP), focusing on text preprocessing, feature extraction, topic modeling, and text classification.

---

## 🚀 Features

- Text preprocessing techniques
- Feature extraction (BoW, TF-IDF)
- NLP utilities using NLTK & Pandas
- Topic Modeling (LDA, LSA)
- Sentiment Analysis
- End-to-end Text Classification pipeline

---


## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/text-classification-nlp.git
cd text-classification-nlp
```

Install dependencies:

```bash
pip install numpy pandas nltk scikit-learn spacy
python -m spacy download en_core_web_sm
```

## ▶️ Usage

Run any notebook using Jupyter:
